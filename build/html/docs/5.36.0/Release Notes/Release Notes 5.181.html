Payara Server 5.181 Release Notes

Release Highlights

This release has been based on GlassFish Server Open Source 5 which was released on September 21, 2017.

Important Information

  • Minimum Required JDK version is now 8u162.

  • Currently, you cannot drop-in and use a Payara 4 domain.xml with Payara 5.

Supported APIs and Applications

  • Java EE 8 Applications

  • MicroProfile 1.2 API
